Weather App

Minimalistic weather app created using HTML, CSS and Javascript. You can search for any city in the world, and the UI will update with information about the weather conditions in the city: day or night time, weather description and air temperature.

To create the app, I made use of:

  • The Accuweather API to search for a matching city and retrieve weather data.
  • The Fetch API with async/await functions to fetch data from Accuweather servers.
  • Bootstrap for styling with a bit of custom CSS-
  • Literal strings to dynamically set img attributes based on data from Accuweather API
