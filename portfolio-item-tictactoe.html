Tic Tac Toe Game

Fun little tic-tac-toe game to play against a friend! Built with HTML, CSS and Javacript.

The challenge when building a game like this is to create a reliable program flow that can keep track of game events as they unfold and trigger appropriate responses when necessary.

To do this, I had to learn how to make use of some of the following methods:

  • data-attributes to keep track cells
  • spread operator to utilize array methods on objects
  • adding and removing event listeners to control game flow
