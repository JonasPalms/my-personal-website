To-do list App

A to-do-list in which you can add new to-dos, delete existing to-dos and filter search for your to-dos

To create the app, I used some of the following techniques and methods:

  • Bootstrap for styling
  • filter() method to search in existing Textcontent properties
  • Dynamically adding new HTML elements based on user input
  • Event delegation to attach event listeners efficiently to both new and existing tasks
