Landing Page UI

Three-page static site made as a mock-up of the Spotify website UI. Contains a landing page, a Discover page and a Join page.

The website makes use of CSS flexbox and psuedo-elements to position the content and graphical elements. To make the site responsive and mobile-friendly I've used media queries at standard breakpoints.
